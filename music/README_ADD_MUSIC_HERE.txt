Optional portable music library.

Library layout:
  music/background/  MP3 or WAV tracks available inside the Music page.
  music/sfx/         MP3 or WAV effects available inside the SFX Library page.

Use the Music page to import, rename, delete, preview, and select a default
track. The SFX page provides the same management actions for effects used by
PLAY markup.
Only use audio whose license permits your intended publication.
