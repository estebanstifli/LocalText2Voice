# LocalText2Voice v0.4.2

Chatterbox CUDA diagnostics and runtime packaging update.

## Fixes

- Added a GPU detector in Chatterbox settings before runtime download.
- The detector shows NVIDIA GPU model, VRAM, driver version, compute
  capability, and driver CUDA support using `nvidia-smi`.
- Added a Chatterbox runtime CUDA diagnostic using PyTorch so users can see
  whether the installed runtime really sees CUDA.
- Fixed the Chatterbox build script so it forces `torch 2.6.0+cu126` and
  `torchaudio 2.6.0+cu126` after installing Chatterbox dependencies.
- Fixed missing PyInstaller package metadata needed by Transformers.
- The CUDA runtime pack is now distributed as multipart GitHub Release assets
  because the CUDA build is larger than GitHub's per-asset upload limit.

## Install

1. Download `LocalText2Voice-v0.4.2-windows-x64.zip`.
2. Extract the ZIP.
3. Run `LocalText2Voice.exe`.
4. Open **Settings > Voice Generation Engine > Chatterbox**.
5. Check **Hardware acceleration** before clicking **Install**.

For NVIDIA GPUs, leave **Compute device** on **Auto (recommended)**. Auto uses
CUDA when PyTorch can see the GPU and falls back only when the runtime cannot
use CUDA.
