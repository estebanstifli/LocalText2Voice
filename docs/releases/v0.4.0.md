# LocalText2Voice v0.4.0

Engine expansion release for Windows x64.

## Highlights

- Piper remains the default fast offline TTS engine.
- Optional API engines for OpenAI TTS, ElevenLabs, and Azure Speech.
- Optional Kokoro local CPU engine with dynamic model installation.
- Optional Chatterbox local GPU engine with on-demand Python dependencies.
- Chatterbox supports multilingual, English, and Turbo model modes, reference
  audio consent checks, CUDA/Auto/CPU/MPS device options, progress, cancel, and
  removal.
- The base portable app stays smaller because Chatterbox dependencies and model
  assets are installed only when the user clicks Install.
- More robust settings, validation, logs, and tests for multi-engine generation.

## Install

1. Download `LocalText2Voice-v0.4.0-windows-x64.zip`.
2. Verify the SHA-256 checksum if desired.
3. Extract the ZIP.
4. Run `LocalText2Voice.exe`.
5. Use **Settings > Voice Generation Engine** to pick Piper, Kokoro, Chatterbox,
   or an API engine.

Voice models and optional large engine assets are not preinstalled because each
provider can have its own license, model card, package size, and hardware
requirements.
