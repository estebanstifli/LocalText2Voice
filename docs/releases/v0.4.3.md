# LocalText2Voice v0.4.3

Follow-up hotfix for Chatterbox CUDA runtime updates.

## Fixes

- Existing Chatterbox runtime installs are now checked against the expected
  runtime version.
- If older AppData metadata exists, **Install** refreshes the Chatterbox Python
  dependencies instead of reusing stale CPU-only metadata.
- The settings UI now marks an existing Chatterbox runtime as **Update
  available** when it is present but outdated.
- Chatterbox settings labels were added to the UI translation files.

## Included from v0.4.2

- GPU detector in Chatterbox settings before runtime download.
- NVIDIA model, VRAM, driver, compute capability, and driver CUDA support.
- PyTorch CUDA runtime diagnostic after Chatterbox is installed.
- CUDA-capable PyTorch dependency validation after install.
