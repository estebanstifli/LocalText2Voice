# LocalText2Voice v0.3.0

First public portable release of LocalText2Voice for Windows x64.

## Highlights

- Offline neural text-to-speech with Piper and no GPU requirement.
- Long-form text chunking, chapter detection, and natural paragraph pauses.
- Built-in multilingual voice browser, remote previews, and validated
  background downloads.
- Clean MP3 narration plus optional podcast mixes with intro, looped background
  music, outro, fades, ducking, and -16 LUFS normalization.
- Responsive PySide6 interface with progress, elapsed time, ETA, cancellation,
  logs, and ten UI languages.
- Portable folder distribution with Piper and FFmpeg kept outside the main
  executable.

## Install

1. Download `LocalText2Voice-v0.3.0-windows-x64.zip`.
2. Verify the SHA-256 checksum if desired.
3. Extract the ZIP.
4. Run `LocalText2Voice.exe`.
5. Download a voice from **Manage voices**.

Voice models are not preinstalled because each voice can have its own dataset
and redistribution terms.
